## Getting Started

### 1. Fork the Repository

Click â€œForkâ€ at the top right of the repository page and clone your fork:

```bash
git clone https://github.com/YOUR_USERNAME/life-api-fastapi.git
cd life-api-fastapi
```

### 2. Create a Branch

Use meaningful names:

```bash
git checkout -b feat/add-new-page
```

### 3. Make Your Changes

Follow the coding style of the project. For Angular:

- Use **TypeScript** best practices
- Follow **Angular style guide**
- Use **TailwindCSS** utility-first approach
- Keep components, services and models modular and testable

### 4. Run the App Locally

```bash
npm install
npm run start
```

### 5. Commit Your Changes

Write clear commit messages:

```bash
git add .
git commit -m "feat(page): add new event creation page"
```

### 6. Push and Open a Pull Request

```bash
git push origin feat/add-new-page
```

Then go to GitHub and open a Pull Request from your branch.
