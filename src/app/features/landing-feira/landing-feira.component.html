Perguntas Frequentes

Que formato de arquivo aceita?

Aceita arquivos TXT com suas anotações de feira no formato simples (data + produtos + valores). Também aceita CSV gerado pelo próprio sistema para reimportar.

Quantos arquivos posso enviar?

Pode enviar múltiplos arquivos de uma vez (ex: 4 feiras do mês). O sistema consolida tudo em um único painel.

Meus dados ficam salvos?

Por padrão, não. Você usa em "sessão privada" e pode apagar tudo ao fechar. Se quiser, pode optar por salvar localmente no seu navegador para reabrir depois.

Funciona offline?

Após carregar a página uma vez, funciona offline. Todo o processamento é local, não precisa de internet.
